快速操作指南

常用操作的快速指南

添加新内容
  1. 点击对应的"添加新XXX"按钮
  2. 在GitHub中输入文件名（如：2024-01-15-news-title.md）
  3. 复制对应的模板内容
  4. 修改模板中的信息
  5. 点击"Commit new file"保存
编辑现有内容
  1. 点击"管理现有XXX"按钮
  2. 在文件列表中找到要编辑的文件
  3. 点击文件名进入详情页
  4. 点击编辑按钮（铅笔图标）
  5. 修改内容后提交更改
删除内容
  1. 进入要删除的文件页面
  2. 点击删除按钮（垃圾桶图标）
  3. 确认删除操作
  4. 填写提交信息
  5. 点击"Commit changes"确认
重要提示
  • 每次修改后，网站会自动重新构建，通常需要2-5分钟生效
  • 建议在修改前先备份重要内容
  • 文件名请使用英文和数字，避免使用特殊字符
  • 图片文件请先上传到 static/images 目录，然后在内容中引用
  • 如有问题，可以查看 提交历史 进行回滚
