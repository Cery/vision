如何使用 GitHub 管理内容

由于静态网站的特性，我们推荐使用 GitHub 直接编辑内容文件：

  1. 点击下方的文件链接，直接在 GitHub 上编辑
  2. 编辑完成后提交更改（Commit changes）
  3. 网站将自动重新构建并更新内容
