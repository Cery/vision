关于 GitHub 内容管理

GitHub 内容管理允许您直接通过 GitHub 仓库编辑网站内容。所有更改都会自动同步到网站，并且支持版本控制和协作编辑。

本地版 GitHub CMS 推荐开发使用

适用场景：个人开发、内容创作
特色功能：实时预览、离线编辑、自动保存、快速响应
使用条件：需要本地Hugo环境 (localhost:1313)
