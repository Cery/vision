操作演示

如何添加新内容
  1. 点击对应的"添加XXX"按钮
  2. 在GitHub页面输入文件名
  3. 复制并修改内容模板
  4. 点击"Commit new file"保存
  5. 等待2-5分钟网站自动更新
文件名建议格式：2024-01-15-title.md
如何编辑现有内容
  1. 点击"管理现有XXX"按钮
  2. 在文件列表中找到目标文件
  3. 点击文件名进入详情页
  4. 点击编辑按钮（铅笔图标）
  5. 修改内容后提交更改
建议编辑前先备份重要内容
